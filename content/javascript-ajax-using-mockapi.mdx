Its time that we move our data to cloud based storage.


# Lets get started

1. Visit [Mockapi.io](https://mockapi.io/), and Sign Up by clicking on the Get Started.

	![Untitled.png](/images/14fae7ce-e14d-4709-adfa-29a7951b3acd.png)

2. You can either sign in using Github, Google or Sign up the traditional method that is my manualing entering the user details.

	![Untitled.png](/images/58c4a1df-9130-4a0b-a3e0-2f6e7fd0f5a1.png)

3. Now that we are inside the MockAPI let first create a project by clicking on the + icon.
4. Enter the name of your project and click on create.

	![Untitled.png](/images/ee7f6173-4a44-4529-a7f1-584df2e702ce.png)

5. Once the project is create click the project name.
6. Lets create a Resource where the datas will be stored and read from by clicking on New Resource.
7. Enter the name for the resource.
	1. Note: if you are going to store user list then name it as users or user_list.
	2. In my case, I am going to store only the tasks.
	3. If have a clear idea of the JSON data you are going to store then You can modify the schema based on your JSON Object

	![Untitled.png](/images/a66c7f67-fb6c-4127-a9b2-2ba902081b24.png)

8. Click on create to create your resource.

	![Untitled.png](/images/8989d20a-dfd9-4c7e-a0fc-ea8878bed1d8.png)

9. As a feature of MockAPI, You can generate data by just clicking on the resource length.

	![Untitled.png](/images/3242c310-3635-4e96-a168-30dc48d2811b.png)

10. Click the Data to see if your randomly genereated data.

	![Untitled.png](/images/3284de39-d78c-4401-82b2-decdd70837fb.png)

11. Now that we have a place to store or retrieve data. We still need a way to access it from our JavaScript project.
12. To solve this  MockAPI provides us a API Url using which we can send and receive data from our MockAPI resource.
13. Copy the URL as below shown

	![Untitled.png](/images/c3f34470-c022-44cd-9d6a-7d58be33a2a0.png)

14. Paste the url in the new tab and replace `:endpoint` with your resource name.

	Note: recommended you install [this](https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh) Extension.


	![Untitled.png](/images/e105d0d8-de15-44df-9915-b9e9295f2571.png)


# Integration

1. Open you project using your IDE. I’ll be using Visual Studio Code

	![Untitled.png](/images/ae12bceb-db3e-45c1-b5a8-792e925e8d0c.png)

2. Using the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) Extension i am able to run my application on [http://localhost:5000](http://localhost:5000/)

	![Untitled.png](/images/6a809214-6c07-4ce3-b6fb-38a16813ca95.png)

3. Now that the application is up and running. We’ll be using the [Axios](https://axios-http.com/docs/intro) library to send and retrieve data the MockAPI URL.
4. copy and paste this script tag in you `.html` file

	```javascript
	<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
	```

5. Lets write a function that will retrieve data from the MockAPI.

	![Untitled.png](/images/d4fb65ba-2db5-45e1-9c10-f401f301ce5e.png)

6. Now that we are able to get data from the API. lets write some code to send data to the API so that we can store new data.

	```javascript
	function sendTaskToURL() {
	
	    const data = {
	        task: "sending data to mockAPI"
	    }
	
	    const url = "https://your-mockapi-endpoint/resource-name";
	    axios.post(url, data)
	        .then(function (response) {
	            console.log(response.data);
	            // TODO: things to happen after the data is sent to the MockAPI
	
	        })
	        .catch(function (error) {
	            console.log(error);
							// TODO: things to happen if failed to send data to the MockAPI
							
	        });
	
	}
	
	sendTaskToURL();
	```
