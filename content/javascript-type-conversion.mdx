Type conversion is the process of converting data of one type to another. For example: converting `String` data to `Number`.


There are two types of type conversion in JavaScript.

- **Implicit Conversion** - automatic type conversion
- **Explicit Conversion** - manual type conversion

# Implicit Conversion


In certain situations, JavaScript automatically converts one data type to another (to the right type). This is known as implicit conversion.


### Conversion to String


```javascript
// numeric string used with + gives string type
let result;

result = '3' + 2; 
console.log(result) // "32"

result = '3' + true; 
console.log(result); // "3true"

result = '3' + undefined; 
console.log(result); // "3undefined"

result = '3' + null; 
console.log(result); // "3null"
```


### Conversion to number


```javascript
// numeric string used with - , / , * results number type
let result;

result = '4' - '2'; 
console.log(result); // 2

result = '4' - 2;
console.log(result); // 2

result = '4' * 2;
console.log(result); // 8

result = '4' / 2;
console.log(result); // 2
```


# Explicit Conversion


You can also convert one data type to another as per your needs. Manual type conversion is known as Explict Conversion.


In Javascript, Built-in methods are available for Explicit Conversion.


## Conversion to Number

- **Number()**
- **parseInt()**
- **parseFloat()** -

### Number()


`Number` returns a number, converted from its argument 


```javascript
let result;

result = Number("3.14");
console.log(result);      // 3.14

result = Number("");
console.log(result);      // 0

result = Number(" ");
console.log(result);      // 0

result = Number("12 34");
console.log(result)       // NaN

```


### parseInt()


`parseInt` method  parses a string and returns an integer.

- If the first character cannot be converted, `NaN` is returned.
- Leading and trailing spaces are ignored.
- Only the first integer found is returned.

```javascript
let result;

result = parseInt("3.14");
console.log(result);      // 3

result = parseInt("");
console.log(result);      // NaN

result = parseInt(" ");
console.log(result);      // NaN

result = parseInt("12 34");
console.log(result)       // 12

result = parseInt("10");
console.log(result);      // 10

result = parseInt("10.00");
console.log(result);      // 10

result = parseInt("10.33");
console.log(result);      // 10

result = parseInt("34 45 66");
console.log(result);      // 34

result = parseInt(" 60 ");
console.log(result);      // 60

result = parseInt("40 years");
console.log(result);      // 40

result = parseInt("He was 40");
console.log(result);      // NaN
```


### parseFloat()


`parseFloat` method parses a string and returns a floating point number.

- If the first character cannot be converted, `NaN` is returned.
- Leading and trailing spaces are ignored.
- Only the first number found is returned.

```javascript
let result;

result = parseFloat("3.14");
console.log(result);      // 3.14

result = parseFloat("");
console.log(result);      // NaN

result = parseFloat(" ");
console.log(result);      // NaN

result = parseFloat("12 34");
console.log(result)       // 12

result = parseFloat("40.00");
console.log(result);      // 40

result = parseFloat(" 40 ");
console.log(result);      // 40

result = parseFloat("40 years");
console.log(result);      // 40

result = parseFloat("40H")
console.log(result);      // 40

result = parseFloat("H40");
console.log(result);      // NaN
```


## Conversion to String


To convert other data types to strings, you can use either

- **String()**
- **toString()**

both the method works in the same way and return the same result when used.


### String()


```javascript
//number to string
let result;
result = String(324);
console.log(result);  // "324"

result = String(2 + 4);
console.log(result); // "6"

//other data types to string
result = String(null);
console.log(result); // "null"

result = String(undefined);
console.log(result); // "undefined"

result = String(NaN);
console.log(result); // "NaN"

result = String(true);
console.log(result); // "true"

result = String(false);
console.log(result); // "false"
```


### toString()


```javascript

let result;

result = (324).toString();
console.log(result); // "324"

result = true.toString();
console.log(result); // "true"
```
